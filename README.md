Dada
====

Dada generates tabular text data.


Usage
-----

    Usage: dada [-d delimiter] [-s size] [-v]


Installation
------------

    make install

The default installation location is `/usr/local`. Override it with `$PREFIX`:

    PREFIX=$HOME make install
