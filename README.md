Dada
====

Dada is a simple program for generating tabular text data.


Installation
------------

    make install

The default installation location is `$HOME`. Override it with `$PREFIX`:

    PREFIX=/usr/local make install


Usage
-----

    Usage: dada [-c columns] [-d delimiter] [-s size] [-v]
