Dada
====

Dada generates tabular text data.


Usage
-----

    Usage: dada [-f field-separator] [-s size]


Installation
------------

Install to `/usr/local/bin`:

    make install

Install to `$HOME/bin`:

    PREFIX=$HOME make install
